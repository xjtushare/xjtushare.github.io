20. 有效的括号

  • 栈解决的问题：
  • 最近相关性
  • 先来后到，先进后出问题。
  • 比较复杂的算法最后都归结到找最近的重复性
  • 因为只能写 if else; for loop 和recursion

解法：
1.暴力
：不断replace匹配的括号

  • 情况： a. ()[]{}
  • b. (([])) 找到匹配的替换成空
  • o(n平方)

2.使用栈

  • 如果是左括号，就压入右括号到栈里面去；

  • loop

  • 如果是右括号，就和栈顶元素进行匹配，如果匹配，就把栈顶元素移除栈，消除这对括号

  • 不然就不合法。

  • if 左括号，入栈

  • else if 右括号，则和栈顶元素进行比较，看是否匹配

  • if 匹配，栈顶元素出栈，栈中元素减少一个，同时字符串的index跳到下一个字符

  • 最后要判断下栈里面的所有元素都清空了，且每一次扫描的字符不会和栈顶元素匹配不上
